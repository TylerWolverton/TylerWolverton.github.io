Project Purple

2017 - Present

Project Purple is a prototype for a 2D multiplayer action RPG that I'm currently developing. I hope to create an intriguing world packed with secrets that can be explored with up to 4 players.

I'm building the game enirely on my own and developing the engine using SDL2 and OpenGL. I've done some work in the past with game programming, but I've never developed engine components, so I wanted to get some hands on experience developing the guts of an engine. I've created an entity component system that uses json files as input for all the actors and levels, making it easy to tweak game actors without recompiling everything. Now that I have most of the main engine functionality completed I can start building the rest of the game and should have something cool to show off soon!
